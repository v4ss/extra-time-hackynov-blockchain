2 - Déploiement du challenge

A chaque fois, vous aurez une description du challenge vous disant ce qu'il faut faire pour le réussir et l'envoyer pour vérification. Ensuite, il vous sera donné le code du smart contract à exploiter afin que vous l'analysiez et que vous trouviez comment atteindre l'objectif attendu. Le plus souvent, le but est de devenir owner du contrat, c'est à dire avoir son adresse de wallet inscrit dans la variable owner du contrat.

Le contrat sera fourni comme ceci :

                        
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract TutoBlockchain {
    address private s_owner;

    constructor() {
        s_owner = msg.sender;
    }

    function getOwnership() external {
        s_owner = msg.sender;
    }

    function getOwner() external view returns(address) {
        return s_owner;
    }
}

Il y aura en dessous du code deux boutons. Un pour déployer une instance de ce contrat (sur laquelle vous pourrez faire ce que vous voulez) et un deuxième bouton permettant de vérifier si vous avez bien atteint l'objectif. Si c'est bien le cas, alors le flag vous sera transmis.

ATTENTION : Lorsque vous faites vérifier votre instance du challenge, veuillez à bien accepter les deux transactions qui s'afficheront l'une après l'autre sur votre Metamask.

Il est conseillé d'avoir la console du navigateur ouverte sur chaque challenge. C'est par ici que vous aurez vos messages d'erreur ou de réussite.

4 - Remix

Remix est un IDE en ligne pour les smart contracts. Il vous permet de déployer, charger, intéragir très simplement avec des smart contracts.

Une fois que vous avez déployé votre instance, son adresse vous sera donnée dans la console du navigateur et sur la page du challenge. Vous la récupèrerez pour pouvoir intéragir avec le contrat.

Pour se faire, ouvrez Remix, créez un nouveau contrat dans votre zone de travail à gauche sous le répertoire "contracts" : Nommez-le <ce_que_vous_voulez>.sol (.sol veut dire que le fichier est dans le langage Solidity, le langage des smarts contracts Ethereum). Collez le code fourni dans le challenge et compilez-le en faisant la combinaison de touche CTRL+S.
Allez ensuite dans l'onglet "Deploy & run transactions" tout à gauche, sélectionnez l'environnement "Injected Provider - Metamask" et connecter votre wallet à Remix.
Sélectionnez bien sur Metamask le réseau de test Sepolia, puis collez l'adresse de votre instance de contrat dans l'input intitulé "At Address" plus bas dans le même onglet de Remix, puis cliquez sur "At Address".
Vous avez chargé votre instance, elle se trouve juste en dessous et vous pouvez maintenant appelez les fonctions du contrat !
